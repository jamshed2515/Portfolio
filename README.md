# Md Jamshed Alam — Portfolio

## How to use

- Open `index.html` in your browser (double-click), **or** run a local server.

### Run a local server (recommended)

If you have Node.js installed:

```bash
npx serve
```

Then open the URL it prints.

## Add your photo

- Put your photo here: `assets/profile.jpg`
- Recommended: square image (e.g., 800×800), JPG or PNG.

## Add your project links

Edit `index.html` and replace the current placeholder buttons under the **Projects** section:

- GitHub repo link for each project
- Live demo link (if hosted)
